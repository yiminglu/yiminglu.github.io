My Vim config

Vim (a contraction of Vi IMproved) is free and open source software and is a widely-used editor.
Nowadays, almost all Vi is soft-linked to vim so that we can type vi to use vim.

Vim’s beauty.

Part of Vim’s power is that it can be extensively customized. The basic interface can be controlled by the many options available, and the user can define personalized key mappings—often called macros—or abbreviations to automate sequences of keystrokes, or even call internal or user defined functions.

There are many plugins available that will extend or add new functionality to Vim. These complex scripts are usually written in Vim’s internal scripting language vimscript. Vim also supports scripting using Lua, Perl, Python, Racket, Ruby, and Tcl.

I’ve reprinted one website (origin) telling many settings and plugins for vim. Take anything you want after you learn it the hard way!!

Vim Pathogen

The Vim Pathogen : Manage your runtimepath with ease. In practical terms, pathogen.vim makes it super easy to install plugins and runtime files in their own private directories.

  • Install to ~/.vim/autoload/pathogen.vim.

  • Add this to your ~/.vimrc.

After all this done, all plugins can be installed in subdirectory under ~/.vim/bundle, so that we can use git to pull all plugins under this directory.
